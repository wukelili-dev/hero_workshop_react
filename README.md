# 勇者工坊 Hero Workshop

放置类 RPG，React + TypeScript 重写版。

## 技术栈

- React 18 + TypeScript
- Vite
- Zustand（状态管理）
- Tailwind CSS
- Radix UI（Dialog / Tabs / Tooltip / Switch / Collapsible / DropdownMenu / Select / Progress / Separator）
- framer-motion（动画）
- react-countup（数字滚动）
- sonner（Toast 通知）

```bash

```
