# 善恶值 + 派系系统 — 实现规划

## 阶段一：核心系统骨架

### 任务1：善恶值 + 派系亲和度（已委托代可行）
- `useGameStore.ts`：添加 moralValue / factions state + changeMoral() / getMoralLevel()
- `Combat.ts`：击杀怪物按类型增减善恶值；calculateDamage() 加派系加成
- `saveUtils.ts`：存档包含善恶值和派系亲和度
- `types.ts`：添加 MoralLevel 类型 / Factions 类型 / Monster.npcType 字段

### 任务2：personalItem 字段 + 30个新NPC数据
- 现有22个NPC补充 personalItem 字段
- 新增12个妖族NPC + 8个仙族NPC
- 详见「NPC数据表」章节

### 任务3：偷窃系统重构
- 移除金币偷窃
- 偷窃成功率 = min(5% + level×0.8% + dex×0.3%, 20%)
- 成功：必得 personalItem + 20%概率附赠小额金币
- 失败：进入 challengeNpc 战斗，善恶值 -15

## 偷窃系统重构规格

### 成功率公式
```typescript
const stealSuccessRate = Math.min(
  0.05 + heroLevel * 0.008 + heroDex * 0.003,
  0.20  // 上限20%
);
```

### 偷窃成功
```typescript
{
  item: npc.personalItem,  // 必得物品
  goldBonus: Math.random() < (npc.stealGoldChance ?? 0.2)
    ? randomInt(npc.stealGoldMin ?? 5, npc.stealGoldMax ?? 20)
    : 0
}
```

### 偷窃失败
- 进入 npc.challengeNpc 战斗
- 善恶值 -15

### npcs.ts 新增字段
```typescript
interface Npc {
  // ... 现有字段
  personalItem?: {
    name: string;
    icon: string;
    description: string;
    sellPrice?: number;  // 向商店出售价格
    effect?: string;    // 效果描述
  };
  stealGoldChance?: number;   // 偷窃成功时附赠金币概率
  stealGoldMin?: number;      // 附赠金币最小值
  stealGoldMax?: number;      // 附赠金币最大值
  moralDialogues?: {
    high?: string[];     // 善恶≥50时用
    neutral?: string[];  // 善恶在-49~+49时用
    low?: string[];       // 善恶≤-50时用
  };
}
```

## 善恶值对话变体（阶段二）

### 袁守城
- **高善（≥50）**：「先生功德无量，龙王冤魂不敢近身。西北有妖龙作祟……」
- **中立（-49~+49）**：「在下观先生面相，一时难断吉凶。」
- **大恶（≤-50）**：「阁下周身死气环绕……在下不便多言。」（部分对话不可用）

### 崔珏
- **高善（≥60）**：「阁下阳寿尚足……这笔我替您收着，待那天再来取。」（赠判官朱笔）
- **中立（-59~+59）**：「判官笔定生死，但我更愿替活人多写几笔福禄。」
- **大恶（≤-60）**：「生死簿上阁下之名已被朱笔圈了三圈。」（警告性对话）

### 敖广
- **高善（≥50）**：「贵客临东海！老夫备了珊瑚露一杯。」
- **中立（-49~+49）**：「泾河那边的事，听说了吧。」（标准对话）
- **大恶（≤-50）**：「来我东海做什么？讨打的吗？」

## 彩蛋（阶段三）

### 彩蛋1：泾河龙王冤案
- 条件：同时持有「占卜铜钱」+「斩龙剑柄碎片」+「泾河令牌」
- 触发：在袁守城处对话自动检测
- 效果：泾河龙王冤魂出现作为可挑战Boss
- 击败掉落：龙魂内丹（HP+500，对龙伤害+50%）

### 彩蛋2：天机不可泄露
- 条件：对神秘老者偷窃连续失败3次以上（成功率<3%时）
- 效果：老者主动送你「天机残卷」，不再触发战斗

### 彩蛋3：赏金猎人认可
- 条件：击败赏金猎人阿飞3次以上
- 效果：阿飞成为「认可」，之后击败野图BOSS额外+5%掉落率

### 彩蛋4：善恶极端值称号
- 善恶值触及+100：称号「至圣」，金光特效，对应派系敌人伤害+30%
- 善恶值触及-100：称号「魔王」，黑气特效，对应派系敌人伤害+30%
